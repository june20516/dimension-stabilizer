# 하드웨어 설계 (두 트랙)

두 트랙을 모두 만들어 비교한다. 공통 PC측은 software-architecture.md 참조.

## Track A — 골전도 트랜스듀서 + 헤드폰 도너 마운트 (전정 자극)

> 개정: 완제품 골전도 이어폰이 아니라, 구매한 **베어 골전도 트랜스듀서(8Ω) ×2**를 **오버이어 헤드폰(도너)** 프레임에 장착한다. 핵심은 트랜스듀서를 두개골 뼈에 **단단히 누르는 것(=클램프력)**.

### A.1 신호 사슬
```
[PC 또는 아두이노+DRV8833: 500Hz] → 트랜스듀서(좌/우)
   → 헤드폰 밴드 클램프력으로 측두골/유양돌기에 압착
게임 오디오 → 스피커(기본 장치)  (별개)
```

### A.2 왜 오버이어 도너인가 (결정 근거)
- 멀미 완화의 1순위 변수는 주파수가 아니라 **압착력(접촉압)**. 골전도는 정적 접촉압이 약하면 진동 전달이 죽는다. (양면테이프만으론 위치는 잡아도 압착이 부족.)
- 헤드폰 밴드의 **스프링 클램프력** = 강하고·일정하고·밴드를 구부려 조절 가능 → 압착 확보에 최적.
- **오버이어**(귀 전체를 감싸는) 쿠션 링은 귀를 둘러싸 **뒤-아래 구간이 유양돌기/측두골을 지난다** → 정상 착용으로도 위치가 맞음. (온이어는 컵이 귀 "위"에 앉아 유양돌기를 비껴감.)
- **정확한 유양돌기 정중앙은 불필요**: 골전도는 두개골 전체가 함께 진동해 전정기관에 전달(임상 VEMP도 유양돌기 또는 이마 Fz에 부착). 현실 타깃 = **귀 뒤·위 단단한 뼈에 단단히 접촉** — 마침 헤드폰이 가장 세게 죄는 측두골이 그 부위.

### A.3 결합 스택 (가장 중요)
```
컵(셸) → 단단한 받침(스탠드오프) → 트랜스듀서 → 얇고 단단한 패드 → 피부/뼈
```
- 그 지점의 **푹신 쿠션은 도려낸다.** 트랜스듀서가 쿠션면보다 **살짝 더 돌출**하게 만들어, 그 돌출부가 클램프력을 **집중 수령**하게 한다. (큰 컵일수록 힘이 링 전체로 퍼져 한 점 압력이 약해지므로 돌출 처리가 필수.)
- 트랜스듀서와 피부 사이에 **푹신한 폼 금지**(진동·힘 죽음). 나머지 쿠션은 편안한 자리잡기 역할만.
- 부착은 **벨크로** → 위치 미세조정·탈착.
- 리드선은 약해 잘 끊김 → 솔더부에 **글루로 스트레인 릴리프**, 밴드 따라 한 곳으로 배출. 좌/우 대칭.

### A.4 도너 선택 기준
- **오버이어(서큐모럴)**, 싼 것(₩5천~1만). 음질·드라이버·코덱·저음 전부 무시.
- 밴드에 **금속심**이 있어 구부려 클램프력 조절 가능한 것 우선(순플라스틱은 구부리면 부러질 수 있음).
- 컵·쿠션 **분해가 쉬운 것**(쿠션 탈착형).

### A.5 구동 (Stage 0 및 이후 공용)
- **Stage 0/기본**: 아두이노 + DRV8833로 500Hz. `tone(AIN1, 500)` + AIN2=LOW, 트랜스듀서는 AOUT1/AOUT2. 사각파라 같은 전압에서 사인보다 에너지↑(힘 테스트 유리). 코드로 250/500/700/1000Hz 스윕.
- **힘 부족 시**: AIN1/AIN2에 180° 반대 위상(BTL 구동) → 진폭 2배.
- VM은 아두이노 5V(USB)로 이 작은 부하엔 충분.
- **노이즈(1–4kHz) 조건**은 아두이노로 만들기 까다로움 → 그건 PC 오디오 경로(`sounddevice`, 별도 출력 장치 지정 + 옵션 PAM8403 앰프)가 쉽다.

### A.6 Stage 0 — 사전 게이트 (헤드폰 가공 전 검증)
"이 소형 트랜스듀서가 두개골 진동을 느끼게 하는가 + 어디가 최적 지점인가"를 0원에 가깝게 먼저 확인.
1. 얇은 **의료용 양면테이프**로 트랜스듀서를 귀 뒤에 임시 부착(또는 손으로 잡고). *두꺼운 폼 양면테이프 금지.*
2. 500Hz 구동(A.5), 250/700/1000Hz 스윕. 좌/우 패닝으로 방향 구분 확인.
3. **손으로 여러 지점(유양돌기·귀 뒤·이마)을 꽉 눌러 비교** → 가장 강하게 느껴지는 위치 = 마운트 목표점 지도.
4. 판단 트리:
   - 느껴짐 → 마운트 제작 진행(클램프는 더 셀 뿐, 안전).
   - 테이프론 약한데 손-압착으론 느껴짐 → **힘 부족**(헤드폰 클램프로 해결).
   - 꽉 눌러도 무감 → **출력 부족** → 더 크고 센 트랜스듀서로 교체.
- **전제**: 이 경로(아두이노+DRV8833)는 Track B 부품이 손에 있어야 실행 가능 → Track B 구매가 Stage 0의 선행조건.

### A.7 폴백 사다리
1. **(1순위)** 오버이어 헤드폰 도너 + 돌출 트랜스듀서. ← 현재 안
2. **(2순위)** 안 쓰는/저가 **골전도 이어폰 프레임 재사용** — 기존 드라이버 들어내고 우리 트랜스듀서 이식. 귀 기준 위치가 안정적이나 적당한 프레임 확보가 관건.

### A.8 BOM (개정)
| 품목 | 사양 | 수량 | 상태 |
|------|------|------|------|
| 골전도 트랜스듀서 | 8Ω 소형(Adafruit #1674급) | 2 | 구매완료(도착 대기) |
| 오버이어 헤드폰(도너) | 싼 것, 금속심 밴드, 쿠션 탈착형 | 1 | 구매 예정 |
| 의료용 양면테이프 | 얇은 피부안전(폼 아님) | 1 | 구매 예정 |
| 단단한 받침·얇은 패드·벨크로 | 스탠드오프/돌출·접착 | 소량 | 구매 예정 |
| (옵션) PAM8403 앰프 | PC 오디오 경로용 | 0~1 | — |

## Track B — 아두이노 + 코인 진동모터 (촉각)

### B.1 신호 사슬
```
[PC] → USB 시리얼 → [아두이노 Uno] → PWM → [트랜지스터/DRV8833] → [코인모터]
```

### B.2 모터 1개 회로 (개별 트랜지스터 방식)
```
아두이노 PWM핀 ─[1kΩ]─ 트랜지스터(2N2222) 베이스
5V ─[코인모터]─ 컬렉터   (모터에 1N4148 플라이백 다이오드 역방향 병렬)
이미터 ─ 공통 GND
```
- 트랜지스터=스위치(핀 직결 시 전류로 보드 손상). 다이오드=역기전력 보호(방향 주의).
- **권장: DRV8833 모듈**(2채널, 플라이백 내장) — 다이오드 방향·배선 실수(초보 최다 에러) 대폭 감소. Track A 구동(A.5)과도 공용.

### B.3 BOM
| 품목 | 사양/검색어 | 수량 |
|------|------------|------|
| 아두이노 | Arduino Uno R3(정품 권장) + USB(Type-B) | 1 |
| 스타터 키트 | 브레드보드·점퍼·트랜지스터(2N2222)·저항(1kΩ)·다이오드(1N4148)·LED 포함 | 1 |
| 모터 드라이버 | DRV8833 2채널 모듈 | 1~2 |
| 코인 진동모터 | "코인 진동모터 3V" ERM | 3~4(리드 약해 여유분) |
| (확장) 외부 전원 | AA 홀더 등 | 0~1 |

### B.4 모터 배치
- 좌/우(yaw) 2모터, **귀 뒤 유양돌기 근처**(전정 가설 부합). 위/아래(pitch) 보류.
- **Track A와 한 리그 공용**: 코인모터를 오버이어 컵에 트랜스듀서 옆(귀 뒤 약간 위)에 글루 → 한 헤드셋으로 OFF/A/B 3way 모두 커버. 촉각이라 위치는 덜 민감.

### B.5 주의
- 다이오드 방향·공통 GND 누락이 초보 최다 실수.
- 3V 모터를 5V로 상시 구동 시 PWM 듀티로 평균 전압↓. 모터 4개 확장 시 외부 전원(공통 GND).
